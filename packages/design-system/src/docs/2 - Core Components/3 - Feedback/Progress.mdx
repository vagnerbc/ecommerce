# **Progress**

O **progress** é usado para exibir o status do andamento de uma tarefa que leva muito tempo ou consiste em várias etapas.

<br />

## Anatomy

O **progress** é formada com base nos Global Tokens.

| background-color                | stroke-color (bar)               | Border Radius             | Exemplo   |
| :------------------------------ | :------------------------------- | :------------------------ | :-------- |
| **`$neutral-color-extralight`** | **`$brand-color-primary-light`** | **`$border-radius-pill`** | _[embed]_ |
