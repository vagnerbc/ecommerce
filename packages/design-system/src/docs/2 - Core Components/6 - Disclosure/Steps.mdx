# **Steps**

**Step** tem como objetivo mostrar para a pessoa usuária a quantidade de etapas dentro de um fluxo, bem como indicar qual etapa está sendo apresentada.

<br />

## Anatomy

Os **Steps** é formado com base nos Global Tokens.

<br />

| Class          | State     | Side (Fill color)                       | Text                    | Text Weight             | Text Color                  | Icon Color                  | Border (Weight / Stroke Color)                              | Exemplo   |
