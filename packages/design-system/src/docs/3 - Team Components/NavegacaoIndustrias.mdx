# **Navegação / Indústrias**

O Tupi, utiliza uma tab personalizada que demos o nome de **Navegação / Indústrias**.

<br />

## Anatomy

A **Navegação / Indústrias** é formado com base nos Global Tokens, e base componentes.

| Class                  | Componente | Componente | Text Style | Exemplo   |
| :--------------------- | :--------- | :--------- | :--------- | --------- |
| Navegação / Indústrias | Base Card  | Badge      | XS Base    | _[embed]_ |

<br />

## Do
