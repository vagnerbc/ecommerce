# **Badges**

Os **Badges** são usados ​​para destacar o status de um item para reconhecimento rápido.

<br />

## Anatomy

O **Badges** é formado com base nos Global Tokens.

| Variant | Container (Fill color)          | Text                    | Text Weight             | Text Color                   | Text-transform | Border Radius           | Border (Weight / Stroke Color)                         | Exemplo   |
| :------ | :------------------------------ | :---------------------- | :---------------------- | :--------------------------- | :------------- | :---------------------- | :----------------------------------------------------- | :-------- |
| Solid   | **`$neutral-color-dark`**       | **`$font-family-base`** | **`$font-weight-bold`** | **`$standard-color-white`**  | uppercase      | **`$border-radius-xs`** | '----'                                                 | _[embed]_ |
| Outline | transparent                     | **`$font-family-base`** | **`$font-weight-bold`** | **`$neutral-color-dark`**    | uppercase      | **`$border-radius-xs`** | **`$border-width-hairline`** **`$neutral-color-dark`** | _[embed]_ |
| Subtle  | **`$neutral-color-extralight`** | **`$font-family-base`** | **`$font-weight-bold`** | **`$neutral-color-darkest`** | uppercase      | **`$border-radius-xs`** | '---'                                                  | _[embed]_ |

<br />

## Do

Use quando a função é somente desacatar.

<br />

## Dont't

Não use em locais que a função é de uma tag, para isso use os componentes Tag Category ou Tag Select.
