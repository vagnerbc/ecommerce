# **Spacing**

Espaçamentos que padronizam a composição e ritmo vertical/horizontal dos componentes e interfaces.

| Name   | Size    | Token                 |
| :----- | :------ | :-------------------- |
| Quarck | 0.25rem | **`$spacing-quarck`** |
| Nano   | 0.5rem  | **`$spacing-nano`**   |
| XXXS   | 1rem    | **`$spacing-xxxs`**   |
| XXS    | 1.5rem  | **`$spacing-xxs`**    |
| XS     | 2rem    | **`$spacing-xs`**     |
| SM     | 2.5rem  | **`$spacing-sm`**     |
| MD     | 3rem    | **`$spacing-md`**     |
| LG     | 3.5rem  | **`$spacing-lg`**     |
| XL     | 4rem    | **`$spacing-xl`**     |
| XXL    | 5rem    | **`$spacing-xxl`**    |
| XXXL   | 7.5rem  | **`$spacing-xxxl`**   |
| Huge   | 10rem   | **`$spacing-huge`**   |
| Giant  | 12.5rem | **`$spacing-giant`**  |

<br />

## Spacing Inset

Espaçamentos internos (**`paddings`**) de um componente onde todos os valores são exatamente iguais.

| Name   | Size             | Token                       |
| :----- | :--------------- | :-------------------------- |
| Quarck | V: 4px; H: 4px   | **`$spacing_inset-quarck`** |
| Nano   | V: 8px; H: 8px   | **`$spacing_inset-nano`**   |
| XS     | V: 16px; H: 16px | **`$spacing_inset-xs`**     |
| SM     | V: 24px; H: 24px | **`$spacing_inset-sm`**     |
| MD     | V: 32px; H: 32px | **`$spacing_inset-md`**     |
| LG     | V: 40px; H: 40px | **`$spacing_inset-lg`**     |

<br />

## Aplicando espaçamento

A escala de espaçamento pode ser aplicada a **`margin`** ou **`padding`**, bem como a arestas verticais e horizontais. O token substitui os valores normalmente atribuídos a **`margin`** e **`padding`**.
