## Do

Os avatares devem ser usados em situações em que seja necessário:

1. Identificar o usuário logado no sistema;
2. Identificar os usuários responsáveis pela adição ou alterações de uma informação;
3. Identificar os usuários responsáveis por uma determinada tarefa.
