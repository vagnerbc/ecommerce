# Design System - E-commerce

Welcome to E-commerce Design System.
**Produto em constante atualização.**

## Organização

Entenda como o **Tupi** está organizado para navegar com maior facilidade entre as especificações.

### DESIGN TOKENS

Os tokens de design são diretrizes que compõe o visual representadas como dados, que garantem experiências de produto sistematicamente unificadas e coesas (espaçamento, cor, tipografia, estilos, etc.).

### BASE COMPONENTS

São os componentes que darão suporte na construção de toda a interface com seu comportamento e interação. Eles são compostos por formas e tokens. Eles são aqueles indivisíveis, as menores partes que formam nossa interface (botões, links, labels, etc.).

### TEAM COMPONENTS

Também são componentes criados a partir dos base componentes, mas consideramos como componentes "especiais" aqueles criados com características exclusivas para um determinado produto.
