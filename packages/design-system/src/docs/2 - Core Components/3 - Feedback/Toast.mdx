# **Toast**

O **toast** é usado para mostrar alertas sobre uma sobreposição. O **toast** se fechará quando o botão Fechar for clicado ou após um tempo limite - **_o padrão é 5 segundos_**. O componente **toast** é usado para fornecer feedback aos usuários após a ocorrência de uma ação.

<br />

## Anatomy

Os **toast** são formados com base nos Global Tokens.

## Do

1. Podem aparecer temporariamente na parte inferior da tela;
2. Utilize o Toast Success para alertas positivos;
3. Utilize o Toast Warning para alertas neutros;
4. Utilize o Toast Error para alertas negativos ou erros;
5. Utilize o Toast Info para alertas informativos;
6. Mantenha a mensagem concisa e clara, evitando colocar mais frases com mais de uma linha no título.
