# **Circular Progress**

O componente **Circular Progress** é usado para indicar o andamento de processos determinados e indeterminados.

<br />

## Anatomy

O **circular progress** é formada com base nos Global Tokens.

| background-color                | stroke-color (bar)               | Text                    | Text Weight                | Text Color                     | Exemplo   |
| :------------------------------ | :------------------------------- | :---------------------- | :------------------------- | :----------------------------- | :-------- |
| **`$neutral-color-extralight`** | **`$brand-color-primary-light`** | **`$font-family-base`** | **`$font-weight-regular`** | **`$neutral-color-extradark`** | _[embed]_ |
