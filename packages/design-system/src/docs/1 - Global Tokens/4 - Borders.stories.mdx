# **Border**

<br />

## Border Width

Variáveis usadas para especificar a espessura de borda dos elementos.

| Width | Name     | Token                        |
| :---- | :------- | :--------------------------- |
| 0px   | None     | **`$border-width-none`**     |
| 1px   | Hairline | **`$border-width-hairline`** |
| 2px   | Thin     | **`$border-width-thin`**     |
| 4px   | Thick    | **`$border-width-thick`**    |
| 8px   | Heavy    | **`$border-width-heavy`**    |

<br />

## Border Radius

Variáveis que definem as bordas de formas e elementos.

| Name     | Radius  | Token                   |
| :------- | :------ | :---------------------- |
| None     | 0px     | $border-radius-none     |
| XS       | 0.25rem | $border-radius-xs       |
| SM       | 0.5rem  | $border-radius-sm       |
| MD       | 1rem    | $border-radius-md       |
| LG       | 1.5rem  | $border-radius-lg       |
| Pill     | 9999px  | $border-radius-pill     |
| Circular | 50%     | $border-radius-circular |
