# **Tag Select**

**Tag Select** serve para selecionar uma categoria de conteúdo, deixando claro que ela pertence a um entre vários tipos.

<br />

## Anatomy

A **Tag Select** é formada com base nos Global Tokens, e faz uso do componente close button.

## Do

A tag select devem ser usados em situações onde seja necessário:

1. Indicar qual é a seleção realizada dentro de um filtro;
2. Quando a pessoa usuário precisar executar alguma ação de click.

<br />

## Dont't

Quando não for preciso uso de ações clicáveis, para isso use o componente tag category.
