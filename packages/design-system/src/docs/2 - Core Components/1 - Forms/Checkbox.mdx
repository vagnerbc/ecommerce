# **Checkbox**

São elementos que possuem uma caixa de seleção que permite ao usuário selecionar ou desmarcar a seleção uma ou mais opções.

## Do

1.  Use as caixas de seleção quando as pessoas usuárias devem selecionar uma ou mais opções de uma lista de itens relacionados;
2.  Pode ser usado para filtrar dados em uma página, em um menu ou em um componente e em uma ação explícita na qual é necessária para aplicar as configurações;
3.  Podem ser utilizados em alguns componentes como: Modais, Formulários, Tabelas, Termos e Condições.
4.  Em uma lista com várias seleções, opte por usar checkboxes. Elas indicam item selecionais ocupando menos espaço.

<br />

## Dont't

1.  Não use checkboxes quando a resposta for apenas uma opção entre duas possíveis. Neste caso, é recomendado o Radio Button.
