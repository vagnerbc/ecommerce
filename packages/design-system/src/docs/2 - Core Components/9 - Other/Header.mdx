# **Header**

**Header** é o menu principal que tem como objetivo ter os links mais importantes que pode interessar a pessoa usuária que entra no site, desta forma, é possível percorrer nas principais categorias e acessar o login para entrar na área logada.

<br />

## Anatomy

O **header** é formado com base nos Global Tokens.

| Type    | Height                     | Device         | Columns | Shandow                      | Background                  | Exemplo   |
| :------ | :------------------------- | :------------- | :------ | :--------------------------- | :-------------------------- | :-------- |
| Default | Desktop: 56px Mobile: 56px | Desktop Mobile | 1 2 3   | **`$shadow-primary-level1`** | **`$standard-color-white`** | _[embed]_ |
| Title   | Desktop: 80px Mobile: 56px | Desktop Mobile | 1 2 3   | **`$shadow-primary-level1`** | **`$standard-color-white`** | _[embed]_ |

<br />

## Do

Sempre será usado no topo das páginas, como navegação principal.
