# **Tabs**

Um componente de **tabs** acessíveis que fornece interações de teclado e atributos ARIA descritos no padrão de design de guias WAI-ARIA.

<br />

## Anatomy

As **tabs** é formada com base nos Global Tokens.

## Do

1. Utilize letras maiúsculas apenas no início da frase apresentada
2. Use labels concisos, fáceis de digitalizar e claros para indicar a ação a ser tomada pela pessoa usuária.

<br />

## Dont't

1. Não utilize maiúsculas em todas as palavras ou letras.
2. Não use labels longos e/ou redundantes, que dificultem a leitura da tab.
