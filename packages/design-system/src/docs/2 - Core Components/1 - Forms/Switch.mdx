# **Switch**

O componente **Switch** é usado como alternativa ao componente Checkbox. Você pode alternar entre os estados ativado ou desativado.

<br />

## Do

Os switch devem ser usados em situações onde os usuários precisam:

1. Ativar ou desativar uma opção de configuração;
2. Indicar seleções de opções binárias, como um sim ou não.

<br />

## Dont't

Os switch não devem ser utilizados quando for necessário selecionar uma ou mais opções dentro de uma lista. Em vez disso, utilize checkbox ou radiobutton.
