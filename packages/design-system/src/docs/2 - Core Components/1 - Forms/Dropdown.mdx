# **Dropdown**

**Dropdowns** apresentam uma lista de opções das quais um usuário pode selecionar uma opção ou várias. Uma opção selecionada pode representar um valor em um formulário ou pode ser usada como uma ação para filtrar ou classificar o conteúdo existente.

<br />

## Do

1. Use para navegação ou menus de comando, onde uma ação é iniciada com base na seleção de um item, ou quando for preciso a aplicação de componentes na lista.

<br />

## Dont't

1. Quando for de apenas uma escolha, para isso use o Select.
