# **Tag Category**

**Tag Category** serve para categorizar um determinado conteúdo, deixando claro que ela pertence a um entre vários tipos.

<br />

## Anatomy

A **Tag Category** é formada com base nos Global Tokens.

<br />

## Do

As tag category devem ser usados em situações onde seja necessário:

1. Indicar qual é a seleção realizada dentro de um filtro;
2. Indicar qual ou quais seleções foram realizadas dentro de uma lista de opções.

<br />

## Dont't

Quando for preciso uso de ações clicáveis, para isso use o componente tag select.
