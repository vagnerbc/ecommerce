# **Icon Button**

São elementos clicáveis ​​usados ​​para **habilitar funções ou atalhos para contextos externos.** Devem conter um ícone que represente com exatidão a ação a ser realizada, sem nenhuma label.

<br />

## Anatomy

O Button Icon é formado com base nos Global Tokens.

## Do

1. Os **`buttons icon`** devem ser usados em situações onde os usuários precisam:
2. Habilitar ações como abrir dropdowns, editar informações, etc.;
3. Atalhos para acessar informações contidas em contextos externos ao qual está inserido;
4. Use em casos onde a área de ação é bastante condensada, ou seja, inviável de se adicionar um botão primário com label.

<br />

## Dont't

1. Confirmar uma alteração ou crIação realizada em um determinado contexto. Em vez disso, prefira **`Buttons`**.
