# Tipografia

O **Tupi** utiliza as famílias de fontes **Nunito** e **Open Sans** licenciadas pelo Open Font License. Elas foram cuidadosamente escolhidas para atender às necessidades dos produtos do ecommerce. Você pode baixá-las gratuitamente em [https://fonts.google.com/](https://fonts.google.com/).

| **Nunito**              | **Open Sans**                |
| :---------------------- | :--------------------------- |
| **Family Base**         | **Family Highlight**         |
| **`$font-family-base`** | **`$font-family-highlight`** |

**Pesos (font-weight)**
Essas são as configurações para definir o quão fina ou pesada é a espessura da fonte.

| **Font Family** | **Font Weight**  | **Name**        | **Token**                     |
| :-------------- | :--------------- | :-------------- | :---------------------------- |
| Nunito          | Regular (400)    | Weight Regular  | **`$font-weight-regular`**    |
| Nunito          | Medium (500)     | Weight Medium   | **`$font-weight-medium`**     |
| Nunito          | Semi-bold (600)  | Weight Semibold | **`$font-weight-semibold`**   |
| Nunito          | Bold (700)       | Weight Bold     | **`$font-weight-bold`**       |
| Nunito          | Extra-bold (800) | Extra-bold      | **`$font-weight-extra-bold`** |
| Nunito          | Black (900)      | Black           | **`$font-weight-black`**      |
| Open Sans       | Regular (400)    | Weight Regular  | **`$font-weight-regular`**    |
| Open Sans       | Medium (500)     | Weight Regular  | **`$font-weight-medium`**     |
| Open Sans       | Semibold (600)   | Weight Semibold | **`$font-weight-semibold`**   |
| Open Sans       | Bold (700)       | Weight Bold     | **`$font-weight-bold`**       |
| Open Sans       | Extra-bold (800) | Weight Bold     | **`$font-weight-extra-bold`** |

**Tamanhos (font-size)**
Essas são as configurações para definir os tamanhos da fonte.
Nossa unidade de medida é o **Rem**, e nossa base é **16px**.

| Name    | Size     | Token                    |
| :------ | :------- | :----------------------- |
| XXXS    | 0.75rem  | **`$font-size-xxxs`**    |
| XXS     | 0.875rem | **`$font-size-xxs`**     |
| XS      | 1rem     | **`$font-size-xs`**      |
| SM      | 1.25rem  | **`$font-size-sm`**      |
| MD      | 1.5rem   | **`$font-size-md`**      |
| LG      | 2rem     | **`$font-size-lg`**      |
| XL      | 2.5rem   | **`$font-size-xl`**      |
| XXL     | 3rem     | **`$font-size-xxl`**     |
| XXXL    | 4rem     | **`$font-size-xxxl`**    |
| Display | 5rem     | **`$font-size-display`** |
| Giant   | 6rem     | **`$font-size-giant`**   |

**Altura da linha (line-height)**
Essas são as configurações para definir o quão baixa ou alta é a altura da linha em que a fonte é encaixada.

| Name    | Size     | Token                      |
| :------ | :------- | :------------------------- |
| Default | 1rem     | **`$line-height-default`** |
| XS      | 1.125rem | **`$line-height-xs`**      |
| SM      | 1.188rem | **`$line-height-sm`**      |
| MD      | 1.313rem | **`$line-height-md`**      |
| LG      | 1.5rem   | **`$line-height-lg`**      |
| XL      | 1.688rem | **`$line-height-xl`**      |
| XXL     | 2rem     | **`$line-height-xxl`**     |
