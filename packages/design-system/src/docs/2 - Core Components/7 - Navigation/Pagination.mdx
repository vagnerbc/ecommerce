# **Pagination**

A **pagination** é usada para dividir conteúdo ou dados em várias páginas, com um controle para navegar para a página seguinte ou anterior.

<br />

## Anatomy

A **pagination** é formado com base nos Global Tokens, e utiliza o componente icon button.
