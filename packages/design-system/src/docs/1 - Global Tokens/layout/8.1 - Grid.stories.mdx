# **Grid**

Grid é a base geométrica de todos os elementos visuais do **Tupi**. Ele fornece estrutura e orientação para todas as tomadas de decisões criativas.

| Name        | Device  | Valores                                                    |
| :---------- | :------ | :--------------------------------------------------------- |
| Fixed XL    | Desktop | Column: 12 / Gutter: 24px / Margin: 0                      |
| Fluid LG    | Desktop | Container: 100% / Column: 12 / Gutter: 24px / Margin: 24px |
| Fluid MD    | Desktop | Container: 100% / Column: 12 / Gutter: 16px / Margin: 16px |
| Mobile Grid | Mobile  | Container: 100% / Column: 4 / Gutter: 20px / Margin: 24px  |
| Fluid SM    | Mobile  | Container: 100% / Column: 12 / Gutter: 8px / Margin: 16px  |

<br />

## Do

Utilize o sistema de grid para compor telas e componentes, respeitando **margens** e **gutters.**

<br />

## Dont't

Não ignore as **margens** e **gutters** dos grids\*\*.
