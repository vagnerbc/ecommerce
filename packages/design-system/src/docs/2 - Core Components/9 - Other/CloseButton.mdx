# **Close Button**

**Close Button** é essencialmente um botão com um ícone de fechar. Ele é usado para lidar com a funcionalidade de fechamento em componentes de feedback e sobreposição, como alerts, toasts, drawers e modals.

<br />

## Anatomy

O **Close Button** é formada com base nos Global Tokens e se faz o uso do componente icon size para controle das dimensões dos ícones..
