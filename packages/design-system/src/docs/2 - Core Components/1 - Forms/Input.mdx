# **Input**

Deve ser utilizado sempre que houver a **possibilidade do usuário inserir informações curtas**, como descrições, resumos, etc.

<br />

## Anatomy

O Input é formado com base nos Global Tokens.

## Do

1. Use em formulários para permitir as pessoas inserirem informações;
2. Podem fazer parte de componentes como modais no caso de um modal de login, por exemplo;
3. Os tipos de entrada de texto comuns incluem: nomes de pessoas usuárias, descrições, URLs, e-mails, endereços e qualquer outra forma de inserção de informação.

<br />

## Dont't

1. Não utilize para entrada de informações longas, para isso utilize textarea.
