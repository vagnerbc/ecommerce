# **SelectSearch**

SelectSearch é um componente que permite aos usuários escolher um valor entre as opções predefinidas. Idealmente, deve ser usado quando houver mais de 5 opções, caso contrário, você pode considerar usar um grupo de rádio.
O SelectSearch permite que o usuário filtre as opções digitando no próprio input e também é possível selecionar multi valores, os valores selecionados ficam em formato de badge no input.

<br />
