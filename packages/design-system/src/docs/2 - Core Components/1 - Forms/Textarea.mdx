# **Textarea**

O **Textarea** permite criar facilmente entradas de texto de várias linhas.

<br />

## Anatomy

O Textarea é formado com base nos Global Tokens.

## Do

Os text areas devem ser usados em situações onde os usuários precisem:

1. Fornecer longos inputs de texto.

<br />

## Dont't

Os text areas não devem ser utilizados como campos de preenchimento para conteúdos curtos. Em vez disso, utilize input.
