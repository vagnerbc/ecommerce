# **Breakpoints**

Utilize os breakpoints padrões para manter a integridade do layout em todos os tamanhos de tela. Para obter melhores resultados, faça testes de design em cada um dos breakpoints.
O Tupi utiliza **`em`** como unidade de medida.

| Class | Width |
| :---- | :---- |
| SM    | 30em  |
| MD    | 48em  |
| LG    | 62em  |
| XL    | 80em  |
| 2XL   | 90em  |
