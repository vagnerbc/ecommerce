## Do

1. Os radio devem ser usados em situações onde os usuários precisam:
2. Selecionar apenas um item dentro de uma lista opções em meio de formulários ou outras situações;
3. Podem ser utilizados em alguns componentes como: Listas, Formulários e Tabelas.

<br />

## Dont't

1. Não deve ser usado se mais de uma opção puder ser selecionada, nesse cenário devemos utilizar o checkbox.
