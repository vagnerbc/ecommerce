# **Tooltip**

Um **tooltip** é uma mensagem breve e informativa que aparece quando um usuário interage com um elemento. As dicas de ferramentas geralmente são iniciadas de duas maneiras: por meio de um gesto de mover o mouse ou por meio de um tab do teclado.

<br />

## Anatomy

O **tooltip** é formado com base nos Global Tokens.

<br />

## Do

Os tooltips devem ser usados em situações onde os usuários precisem:

1. Entender o significado ou propósito de ícones;
2. Acessar uma informação complementar;
3. Ser orientados a realizar alguma ação (ajuda contextual);
4. Receber alguma dica de uso (para uma nova funcionalidade, por exemplo).

<br />

## Dont't

Os tooltips não devem ser utilizados para grandes informações. Em vez disso, prefira o alert dialog.
