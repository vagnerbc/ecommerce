# **Slider**

O **slider** é usado para permitir que os usuários façam seleções de um intervalo de valores.

<br />

## Anatomy

O **slider** é formada com base nos Global Tokens.

| background-color                 | stroke-color (bar)             | Slider Thumb Color                 | Exemplo   |
| :------------------------------- | :----------------------------- | :--------------------------------- | :-------- |
| **`$neutral-color-mediumlight`** | **`$neutral-color-extradark`** | **`$brand-color-primary-default`** | _[embed]_ |

<br />

## Do

Os sliders devem ser usados em situações onde os usuários precisam:

1. Selecionar um valor dentro de um determinado intervalo com valores mínimo e máximo.

<br />

## Dont't

1. Não use para intervalos extremamente grandes (por exemplo, 1-1000).
2. Não use para intervalos muito pequenos (por exemplo, 1-3).
3. Não utilizar quando necessitar de uma escolha de valor específico. A interação do handle deslizante não é tão precisa quanto digitar ou escolher valor em um number input.
