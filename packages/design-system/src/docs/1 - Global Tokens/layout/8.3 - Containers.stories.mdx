# **Containers**

Os contêineres são usados ​​para restringir a largura de um conteúdo ao ponto de interrupção atual, mantendo-o fluido, utilize para compor telas.
O Tupi utiliza **`em`** como unidade de medida.

| Class | Width |
| :---- | :---- |
| SM    | 30em  |
| MD    | 48em  |
| LG    | 62em  |
| XL    | 80em  |
| 2XL   | 90em  |
