# **Opacity**

Variáveis que permitem determinar o nível de opacidade de um elemento.

| Name             | Level | Color Token                          |
| :--------------- | :---- | :----------------------------------- |
| Semi Opaque      | 0.72  | **`$opacity-level-semiopaque`**      |
| Intense          | 0.64  | **`$opacity-level-intense`**         |
| Medium           | 0.32  | **`$opacity-level-medium`**          |
| Light            | 0.16  | **`$opacity-level-light`**           |
| Semi Transparent | 0.08  | **`$opacity-level-semitransparent`** |
