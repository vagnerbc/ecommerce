# **Button**

São elementos clicáveis ​​usados ​​para **executar uma ação importante em páginas, modais e formulários.** Devem conter um _label_ curto que descreva a ação que ocorrerá quando o usuário clicar nele.

## Do

Os buttons devem ser usados em situações onde os usuários precisam:

1.  Confirmar uma alteração realizada em um contexto específico;
2.  Confirmar a criação de uma nova tarefa ou registro;
3.  Navegar entre etapas de um processo.

<br />

## Dont't

Os buttons não devem ser utilizados como elementos de navegação entre jornadas/fluxos. Em vez disso, utilize tabs, breadcrumbs, etc.

<br />

## Labels

1.  O label nunca terá quebra de linha.
2.  O label em botões sempre terá o primeiro caractere em letra maiúscula e os demais caracteres em letra minúscula.
3.  Não utilize maiúscula em todas as palavras ou letras.
