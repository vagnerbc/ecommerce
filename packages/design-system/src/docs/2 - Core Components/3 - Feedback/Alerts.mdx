# **Alerts**

Os **alerts** são usados ​​para comunicar um estado que afeta um sistema, recurso ou página.

<br />

## Anatomy

Os **alerts** são formados com base nos Global Tokens.

## Do

1. Informam a pessoa usuária sobre um processo que um aplicativo executou ou executará;
2. Podem aparecer temporariamente na parte inferior da tela;
3. Utilize o Alert Success para alertas positivos;
4. Utilize o Alert Warning para alertas neutros;
5. Utilize o Alert Error para alertas negativos ou erros;
6. Utilize o Alert Info para alertas informativos;
7. Mantenha a mensagem concisa e clara, evitando colocar mais frases com mais de uma linha no título.

<br />

## Dont't

1. Não deve ser aplicado no meio de algum fluxo;
2. Não deve interromper a experiência da pessoa usuária;
3. Evite empilhar alertas sobre o outro e não mude seu formato.
