# **Typography**

Esses são os estilos títulos e textos predefinidos que aplicamos nas nossas interfaces eles são criados a partir dos tokens definidos nas seções **Tipografia/Font Family/Family Base** ou **Tipografia/Font Family/Family Highlight** seus estilos e valores devem ser mantidos do **Text Style Name**.

<br />
