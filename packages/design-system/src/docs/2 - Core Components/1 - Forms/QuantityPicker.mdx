# **Quantity Picker**

O **Quantity Picker** é utilizado para mensurar quantidades de itens, pode ser usar para produtos ou em outro componente que precise controlar quantidades

<br />

## Anatomy

Os **quantity picker** são formados com base nos Global Tokens.

## Do

Use para controlar quantidade de itens, exemplo quantidade de produtos no carrinho.

<br />

## Dont't

Não use para fazer controle de escala etc.
