# **Icon Sizes**

O Icon Sizes é responsável por manter a harmoniza na padronização das dimensões de ícones do Tupi, utilizamos como biblioteca o Material Icons do Google.

| Class | Size     | Token                 | Exemplo   |
| :---- | :------- | :-------------------- | :-------- |
| XXXS  | 0.75rem  | **`$icon-size-xxxs`** | _[embed]_ |
| XXS   | 0.875rem | **`$icon-size-xxs`**  | _[embed]_ |
| SM    | 1rem     | **`$icon-size-xs`**   | _[embed]_ |
| MD    | 1.5rem   | **`$icon-size-md`**   | _[embed]_ |
| LG    | 2rem     | **`$icon-size-lg`**   | _[embed]_ |

<br />

## Como utilizar o componente de ícone?

**Importante:** Ícones em componentes devem ser utilizados dentro do componente “**Icon Sizes**”.
**_Ref: Figma_**

1.  Insira o componente “**Icon Sizes**” no componente que deseja utilizar.
2.  Agora com o componente selecionado, utilize a barra de propriedades para o tamanho desejado na variante “**Size**”.
3.  Clique no componente, com o componente selecionado clique na layer “**Icon Sizes**” expanda a layer para visualizar a layer “**Troque o ícone aqui**”.
4.  Agora na barra de propriedades clique no nome do ícone, utilize a busca para encontrar o ícone que deseja utilizar.

<br />

## Do

Utilizar o Icon Sizes para inserir ícones em componentes e telas.

<br />

## Dont't

Não utilizar o **`Icon Sizes`** para efeitos visuais, representativos tais como glifos ou representação gráfica, para isso você deve utilizar o SVG base do Material icons.

Não utilizar tamanhos que não estejam definidos na tabela de padronização.
