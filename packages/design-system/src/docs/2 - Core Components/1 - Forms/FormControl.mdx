# **Form Control**

**Form Control** fornece contexto como **isInvalid**, **isDisabled** e **isRequired** para elementos de formulário, com a adição do componente label.

<br />

## Anatomy

O **Form Control** é formado com base nos Global Tokens, e sua base são os componentes de formulário.

<br />
